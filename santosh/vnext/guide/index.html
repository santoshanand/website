Introduction

I did some deep research and benchmarks with ‘wrk’ and ‘ab’ in order to choose which framework would suit me and my new project. The results were really disappointing to me.

I started wondering if golang wasn’t as fast on the web as I had read… but, before I let Google Golang and continued to develop with nodejs.
[13 March 2016] was the day when I said these word to my self.


‘Makis, don’t lose hope, give at least a chance to Golang. Try to build something totally new without basing it off the “slow” code you saw earlier; learn the secrets of this language and make others follow your steps!’.

Birth of Iris Name

The same day (13 March 2016), later the night, I was reading a book about Greek mythology. I saw an ancient goddess’ name and was inspired immediately to give a name to this new web framework (which I had already started writing) - Iris.
